# Nakagin Capsule Tower: Architectural & Review Data

This document contains the parsed spatial data and qualitative review excerpts used to generate the parametric Rhino Python model (`nakagin_generator.py`).

## 1. Hard Spatial Parameters
Based on historical architectural records and visual references:

*   **Total Capsules:** 140
*   **Core A:** 11 stories, ~60 capsules
*   **Core B:** 13 stories, ~80 capsules
*   **Capsule Dimensions:** 2.5m (W) x 4.0m (L) x 2.5m (H)
*   **Iconic Window:** 1.3m diameter, centered on the outer face.
*   **Core Dimensions:** ~4.0m x 4.0m footprint
*   **Metabolist Staggering:**
    *   Lateral Offset: +/- 0.6m
    *   Vertical Offset: 0.0m, 0.4m, 0.8m

## 3. Structural Logic (For Generative Pipeline)
*   **The Spine:** Two interconnected concrete service cores form the primary structure.
*   **The Connection:** Capsules attach via 4 high-tension bolts (designed to be replaceable, though never realized).
*   **The Orientation:** Capsules cantilever, stagger, and rotate organically around the central stems, rejecting a flush, uniform facade.
*   **Materiality:** Lightweight steel-truss boxes covered in galvanized, rib-reinforced steel panels.
